# Customer Churn Analysis

## Overview
This project analyzes telecom customer data to understand why customers leave (churn) and builds a machine learning model to predict customers who are likely to leave.

---

## Goals
- Explore customer behavior
- Identify churn patterns
- Build a churn prediction model
- Suggest retention strategies

---

## Tools Used
- Python (Pandas, Scikit-learn, Matplotlib)
- SQL (data analysis)
- Jupyter Notebook

---

## Project Structure

customer-churn-analysis/

- data → raw and cleaned dataset  
- sql → database schema and analysis queries  
- notebooks → cleaning, EDA, and modeling  
- src → Python scripts  
- visuals → charts and insights  

---

## Key Insights

✔ Month-to-month contracts have the highest churn  
✔ New customers are more likely to leave  
✔ Higher monthly charges increase churn risk  
✔ Electronic check payment users churn more  

---

## Machine Learning Model
A Random Forest model was built to predict customer churn and identify high-risk customers.

---

## Business Recommendations

- Encourage long-term contracts  
- Improve onboarding for new customers  
- Promote automatic payment methods  
- Offer loyalty rewards
